DEVELOPMENT TIPS
================================================================================

Hot Reload:
  • React components auto-reload (Vite)
  • Save files → Changes visible instantly
  • No page refresh needed

Debugging:
  • Browser DevTools (F12)
  • php artisan tinker (database shell)
  • Laravel logs (storage/logs/)

Performance:
  • Lazy-load React components
  • Use Redis for sessions (production)
  • Compress images
  • Cache frequent queries

Code Quality:
  • composer exec pint (format code)
  • Run tests before committing
  • Follow PSR-12 standards

================================================================================
TROUBLESHOOTING
================================================================================

Port 8000 already in use:
  php artisan serve --port 8001

Styles not loading:
  Ensure npm run dev is running

Database connection error:
  Check .env DB_* settings

419 CSRF token error:
  Clear browser cookies

Node modules broken:
  rm -rf node_modules && npm install
