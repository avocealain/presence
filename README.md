# Smart Attendance System (Akıllı Yoklama Sistemi)

A QR-code-based attendance management system designed to streamline presence tracking for educational and professional environments. This application provides a seamless way to log attendance while offering comprehensive analytics and user management tools.

## Live Demo
[https://alainavoce.app](https://alainavoce.app)

## Features
**QR-Based Tracking:** Fast and secure attendance logging using dynamic QR code scanning.
**Admin Dashboard:** A centralized interface for comprehensive user management.
**Real-Time Analytics:** Visualize attendance data, monitor trends, and track user engagement.
**Role-Based Access Control:** Secure authentication ensuring data privacy for both administrators and regular users.
**Responsive Design:** Optimized for both desktop and mobile devices.

## Tech Stack & Architecture
This project leverages a modern web development stack to ensure scalability, performance, and maintainability:

**Frontend:** React
**Backend:** Laravel
**Infrastructure & Deployment:** DigitalOcean
* **Server Management:** Coolify (used for continuous deployment and infrastructure management)
**Security:** Fully secured with automated SSL certification.

## Deployment Details
The application is self-hosted on a **DigitalOcean** droplet. **Coolify** is utilized as the self-hosted PaaS to handle the smooth deployment of both the React frontend and Laravel backend, managing environment variables, seamless updates, and automated SSL configuration for the `alainavoce.app` domain.
